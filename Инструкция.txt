Способ I:
1) Загрузить файл в Oracle согласно присланной видео-инструкции в таблицу с названием "RAWS" (поле 'DATE'- назвать 'DATEE' и тип данных для всех полей выставить VARCHAR2(100)).

2) Открыть DDL файл  --> создать все таблицы(начиная с DIM_TERMINALS и заканчивая Report)  --> создать все триггера приведенные в DDL файле.

3) Открыть ETL файл --> создать все процедуры (начиная с raws_dim_terminals и заканчивая dims_fact_stg_report_20min3o)  -->(3а)  запустить блок кода (BEGIN со всеми перечисленными процедурами END;).
В файле блок кода с корректным порядком запуска процедур находится снизу,запускайте его.

4) Открыть файл Report --> создать процедуру (stg_report_report) -->(4а) запустить процедуру,после чего можно увидеть результат --> SELECT * FROM REPORT order by Fraud_dt.

5) При загрузке следующего файла с сырыми данными штатным методом предусмотренным в Oracle необходимо сделать  --> (Drop table RAWS;) 
   После чего действовать согласно пункту 1 (если использовать сторонний метод по загрузке данных через инкремент в RAWS,то дропать RAWS не надо)для последующего производства запускть ранее созданные процедуры пункты (3а и 4а).
